# Projeto de convidados

## Como rodar 
Baixe o projeto e entre na pasta back-end e rode o codigo
`````
npm i 
npm run dev
`````
ATENÇÃO: crie a env e coloque os dados do seu banco
Entre na pasta frontend 
`````
npm i 
npm run dev
`````

Apos as instalações entre na porta local 5173

O projeto é um consiste em um CRUD de convidados e integrado com o banco de dados tendo um sistema de listagem e forms para o crud 

As tecnologias utilizadas foram para o backend typescript typeorm reflect-metadata express ts-node-dev 

As tecnologias utilizadas foram para o frontend vite react tailwindcss react-router-dom
